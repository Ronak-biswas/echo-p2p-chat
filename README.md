# ECHO - Secure P2P Chat

ECHO is a secure peer-to-peer communication web application.

## Features
- Peer to Peer messaging
- Video calling
- Secure WebRTC connection
- Contact management
- Browser notifications
- Modern UI

## Technologies Used
- HTML
- JavaScript
- TailwindCSS
- PeerJS
- WebRTC

## How to Use
1. Open the website.
2. Enter a unique username.
3. Add your friend's username.
4. Connect and start chatting.
